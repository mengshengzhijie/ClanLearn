### 正数移位操作符

![](Screenshot/Screenshot_20241121_155909.png)

```c
int main () {
    int a = 7;
    // 左移操作符
    int b = a << 1;
    printf("%d\n", a);
    printf("%d\n", b);
}
print : 7 
    	14
```

### 负数移位操作符

```c
int main () {
    int a = -7;
    int b = a << 1;
    printf("%d\n", a);
    printf("%d\n", b);
}
print : -7 
    	-14
```

### 右移

![](Screenshot/Screenshot_20241121_164206.png)

右边丢弃，左边补原符号

```c
int main () {
    int a = 7;
    int b = a >> 1;
    printf("%d\n", a);
    printf("%d\n", b);
}
7 3
```
