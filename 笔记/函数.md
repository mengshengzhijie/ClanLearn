### 递归实现strlen

```c
int my_strlen(char *str) {
    if (*str != '\0') {
        return 1 + my_strlen(str + 1);
    } else {
        return 0;
```

```c

    if (n <= 2) {
        return 1;
    } else {

    }
}
```

### 迭代实现斐波那契

![斐波那契](Screenshot/Screenshot_20241113_170405.png)

```c
int Fib(int n) {
    int a = 1,b = 1,c = 0;
    while (n >= 3) {
        c = a + b;
        a = b;
        b = c;
        n--;
    }
    return c;
}
```
